# aosp2yolo

you can use this to convert
[AvLab/AOSP](https://github.com/AvLab-CV/AOLP)
format to
[AlexeyAB/darknet](https://github.com/AlexeyAB/darknet)
format

```tree
C:.
├─cfg
├─Subset_AC
│  ├─groundtruth_localization
│  ├─groundtruth_recognition
│  └─Image
├─Subset_LE
│  ├─groundtruth_localization
│  ├─groundtruth_recognition
│  └─Image
├─Subset_RP
│  ├─groundtruth_localization
│  ├─groundtruth_recognition
│  └─Image
├─...
```
